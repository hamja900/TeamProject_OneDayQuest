## Project
- 프로젝트 명 :  일일퀘스트 당근을 키워라!

- 프로젝트 소개 :  캐릭터가 떨어지는 물방울을 모아서 당근을 수확하는 게임입니다. 떨어지는 오브젝트에는 물방울, 영양물, 해충 3가지로 구성되어 있으며 난이도는 Easy, Nomal, Hard 입니다. 난이도에 따라 물방울의 떨어지는 속도, 해충이 떨어지는 빈도가 다릅니다.
